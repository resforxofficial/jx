# Tx (formerly Jx)

tx is a small experimental programming language designed around simplicity and readable syntax.

It compiles to JavaScript and focuses on lightweight scripting, clean control flow, and beginner-friendly structure.

Created in Korea by resforxofficial.

<hr/>

## Variable Declaration
mut declares a mutable variable that can be reassigned later.
```
mut int x = 1;
```

immut declares an immutable constant value that cannot be changed after initialization.
```
immut int MAX_NUM = 2;
```
